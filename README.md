🎯 Objective: Configure an Aruba 6300 switch to segment three clients into isolated VLANs, each with DHCP-assigned addressing, while enabling secure inter-client communication.

🛠️ What was used: Aruba 6300 switch, VLANs, DHCP server configuration, static IP routes, SSH key-based authentication, and Wireshark for traffic verification.

💡 What was learned: Gained hands-on experience with VLAN segmentation, subnetting, DHCP pool management, inter-VLAN routing, and secure SSH configuration — along with using Wireshark to verify network traffic at the packet level.
